Спробувати з віртуальним пацієнтом

Введи (або підвантаж із прикладу) JSON-профіль синтетичного пацієнта і отримай повний Plan. Жодних реальних пацієнтських даних — це лише runner для перевірки функціоналу. Рушій (Python) запускається у твоєму браузері через Pyodide.

Перший запуск завантажує Pyodide (~10–15 МБ) та сам рушій. Очікуй ~10–30 секунд при першому запуску, потім <1 с.
