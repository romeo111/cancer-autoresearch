What NOT to do

Explicit prohibitive rules, each grounded in a regimen / supportive care / contraindication entity
Aggressive plan (IND-NK-T-NASAL-1L-P-GEMOX)
  • Не пропускати antithrombin III monitoring при asparaginase — thrombosis.
  • Не давати asparaginase якщо pancreatitis history — fatal pancreatitis.
  • Не пропускати hypersensitivity test перед першою дозою asparaginase.
  • Не пропускати EBV viral load — корелює з disease activity + relapse early detection.
  • Не давати full-dose oxaliplatin при CrCl <30 — dose reduce 50% або switch to alternative.
  • Не комбінувати asparaginase з warfarin без INR monitoring — coagulopathy.
  • Не вводити вінкристин (якщо у subsequent regimen) інтратекально — фатально.
Standard plan (IND-NK-T-NASAL-1L-SMILE)
  • Не давати CHOP — anekдотично ineffective (P-gp-mediated resistance).
  • Не пропускати mesna при ifosfamide — hemorrhagic cystitis.
  • Не пропускати antithrombin III monitoring при asparaginase — thrombosis ризик підвищений.
  • Не вводити вінкристин (якщо у subsequent regimen) інтратекально.
  • Не давати asparaginase якщо pancreatitis history — fatal pancreatitis.
  • Не пропускати EBV viral load — корелює з disease activity + relapse early detection.
